# README

#### Install :

Clone this repository and install dependencies :
```shell
git clone https://github.com/ArnaudRocca/VFA17.git
```
```shell
npm install
```

#### Development :

Start the webpack-dev-server at http://localhost:8080/
```shell
npm start
```

#### Production :

Build a bundled file outputted here : static/bundle.js
```shell
npm run build
```
